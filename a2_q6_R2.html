R^2 = 1 - \dfrac{\sum \hat{e}^2}{ \sum ( y -\bar{y} )^2}

R^2 = 1 - \dfrac{ \frac{1}{n} \sum \hat{e}^2}{ \frac{1}{n} \sum ( y -\bar{y} )^2}

the expression for the numerator is on page 170 of Hansen

\hat{e} = y-x \hat{\beta} = e - x(\hat{\beta}-\beta)

using the expression for e_hat

\frac{1}{n} \sum \hat{e}^2 = \underbrace{\frac{1}{n} \sum {e}^2}_{ \xrightarrow{p} \sigma^2} + A + B [via WLLN, term 1 goes to \sigma^2 and terms 2 and 3 go to 0 ]

therefore \frac{1}{n} \sum \hat{e}^2 \xrightarrow{p} \sigma^2

as for the denominator

from \bar{y} \xrightarrow{p} \mu_y and the WLLN, we get : \frac{1}{n} \sum (y-\bar{y})^2 \xrightarrow{p} \sigma_y^2

R^2 \xrightarrow{p} 1 - \dfrac{ \text{var}(e) }{ \text{var}(y) }

6b

if we use the asymptotic value of R^2

R^2 \rightarrow 1 - \dfrac{\text{var}(e)}{\text{var}(y)}

\text{var}(y) = \text{var}(bx+e) = b^2 \text{var}(x) + var(e) + 2 \text{cov}(x,e)

R^2 \rightarrow 1 - \dfrac{\text{var}(e)}{ b^2 \text{var}(x) + \text{var}(e) + 2 \text{cov}(x,e) }

\text{cov}(x,e) \xrightarrow{p} 0

R^2 \rightarrow 1 - \dfrac{\text{var}(e)}{b^2 \text{var}(x) + \text{var}(e)}

a = 1 - \dfrac{b}{c+b}

\dfrac{\partial a}{\partial c} = (-1)(-b)(c+b)^{-2} = b \cdot (c+b)^{-2} \gt 0

so as \text{var}(x) goes up, R^2 improves asymptotitally
